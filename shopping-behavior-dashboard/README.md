# Shopping Behavior Dashboard (Python Shiny)

## Overview
An interactive multi-page Shiny for Python application designed to explore consumer shopping behavior through dynamic filtering, KPI summaries, and visual analytics.

## Key Features
- Reactive filters by gender, age group, state, and category
- KPI cards for total spend, average purchase value, and transaction counts
- Interactive bar charts and treemaps
- Geographic visualization of customer distribution
- Multi-tab navigation for exploratory analysis

## Technology Stack
- Python
- Shiny for Python
- pandas
- plotly

## Purpose
This application demonstrates the use of reactive programming and dashboard design principles to support data-driven insights into consumer purchasing patterns.
